## 一、图解系列（最适合零基础）

### 1. [ChengFai/CS-Base](https://github.com/ChengFai/CS-Base) ⭐ 零基础首选
**小林图解计算机基础**
- 在线阅读：[xiaolincoding.com](https://xiaolincoding.com)
- **1000+张图解 + 50万字**
- 涵盖模块：
  - 《图解网络》— TCP/IP、HTTP、HTTPS、DNS
  - 《图解系统》— 进程线程、虚拟内存、文件系统、调度算法、IO多路复用

  - 《图解Redis》— 数据结构、持久化、集群
- **特点**：用日常比喻解释技术概念，图文并茂
- **适合**：完全零基础，需要建立直观理解

## 五、零基础入门建议顺序

```
第1步（1小时/天，1周）
  → 看 Crash Course Computer Science（B站有中文字幕）
  → 建立对计算机的整体认知

第2步（1-2个月）
  → 读 小林图解网络 + 图解系统（xiaolincoding.com）
  → 先建立直观理解，不要纠结细节

第3步（2-3个月）
  → 学 C 语言（参考 C语言资源文件）
  → 用 C 写小程序巩固理解

第4步（3-6个月）
  → 学 计算机组成原理（参考你已有的 01.md 课程）
  → 学 操作系统原理（推荐 CS-Notes + MIT 6.S081）
  → 学 数据结构与算法（推荐 严蔚敏 + LeetCode）
```

## 关键提醒

1. **不要一开始就看 CSAPP 这样的硬核书** — 先图解除直观理解
2. **不要跳过计算机组成原理** — 内核开发天天和硬件打交道
3. **不要只看不写** — 每学一个概念至少写一个小程序验证
4. **操作系统是内核的基础** — 不理解操作系统原理，学内核就是空中楼阁
