## 二、必备软件工具

### 开发环境

| 工具 | 用途 | 说明 |
|------|------|------|
| **Ubuntu 22.04/24.04 LTS** | 主力开发系统 | 虚拟机/WSL2安装即可 |
| **VS Code** | 代码编辑 | 配合 C/C++、Remote SSH 扩展 |
| **QEMU** | 内核调试虚拟机 | 安全实验，不影响宿主系统 |
| **Docker** | 构建环境容器 | 可复现的交叉编译环境 |

### 编译工具链

| 工具 | 用途 |
|------|------|
| **aarch64-linux-gnu-gcc** | ARM64 交叉编译器 |
| **GNU Make / CMake** | 构建系统 |
| **Yocto Project** | 高通BSP构建系统 (meta-qcom) |
| **Buildroot** | 轻量级嵌入式Linux构建 |

### 内核调试工具

| 工具 | 用途 | 学习优先级 |
|------|------|------------|
| **GDB** | 内核源码级调试（配合QEMU） | 高 |
| **printk** | 内核日志输出 | 高 |
| **dmesg** | 查看内核环形缓冲区日志 | 高 |
| **ftrace** | 内核函数追踪 | 中 |
| **perf** | 性能分析（CPU/内存/cache） | 中 |
| **bpftrace / eBPF** | 动态追踪（不修改内核） | 中 |
| **KGDB** | 内核远程调试（需JTAG） | 低（进阶用） |
| **kallsyms** | 内核符号表查询 | 高 |

## 三、实验环境搭建速查

### 最简环境（零基础第一天）

```bash
# 1. 装 Ubuntu 22.04 虚拟机或 WSL2
# 2. 安装基本工具
sudo apt update
sudo apt install build-essential git vim gcc gdb cmake

# 3. 克隆第一个内核学习仓库
git clone https://github.com/sysprog21/lkmpg.git
cd lkmpg
# 跟着 README 做第一个内核模块
```

### QEMU 内核实验环境

```bash
# 用 cirosantilli 的自动化脚本
git clone https://github.com/cirosantilli/linux-kernel-module-cheat.git
cd linux-kernel-module-cheat
# 一行命令搭建 QEMU + GDB 环境
./setup
```

### 高通平台交叉编译环境

```bash
# 下载 ARM64 交叉编译工具链
wget https://developer.arm.com/-/media/Files/downloads/gnu/...
# 或使用 apt 安装
sudo apt install gcc-aarch64-linux-gnu

# clone 高通内核源码
git clone https://git.codelinaro.org/clo/la/kernel/msm-5.15.git
cd msm-5.15
export ARCH=arm64
export CROSS_COMPILE=aarch64-linux-gnu-
make msm_defconfig
make -j$(nproc)
```
