# 📚 Linux 内核开发学习笔记

## 课程结构

| 阶段 | 周数 | 内容 |
|------|------|------|
| 环境与基础 | 第1-4周 | Linux 开发环境、C 语言内核特训、Git 工作流、内核编译系统 |
