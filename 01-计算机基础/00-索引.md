# 计算机基础学习路径

## 学习路径概览

### 阶段一：计算机基础
1. 计算机组成原理
2. 操作系统基础
3. 数据结构与算法
4. C语言编程

### 阶段二：Linux基础
1. Linux系统使用
2. Linux命令行
3. Shell脚本
4. Linux系统编程

### 阶段三：Linux内核基础
1. 内核架构与启动
2. 进程管理
3. 内存管理
4. 设备驱动
5. 文件系统

### 阶段四：Qualcomm特定内核开发
1. Qualcomm平台架构
2. SMMU相关知识
3. DPD (Deep Power Down) 技术
4. Proxy机制
5. 调试工具与技巧
