## 当前阶段：Phase 1 — C语言精通 + Linux 系统编程基础

**目标**：能用 C 写出健壮的系统级代码，理解 Linux 用户态编程模型

### Week 1-2：C语言核心

**学什么**：指针、内存布局（栈/堆/BSS/data/text）、结构体、位操作、函数指针、宏
**产出**：通用链表库（参考 Linux kernel `include/linux/list.h`）
**权威资料**：《C程序设计语言》（K&R 第2版）

| 内容 | 状态 | 备注 |

| Linux 基础概念（/proc/权限/进程） | ✅ 完成 | /proc 文件系统、chmod、ps/top |
| C 语言核心 | 🔄 进行中 | — |

### Week 3-4：Linux 系统调用 + 进程/文件

**学什么**：fork/exec/wait、文件描述符、mmap、信号、管道、epoll
**产出**：myshell（支持管道和重定向）

### Week 5-6：多线程 + 同步原语

**学什么**：pthread、mutex、条件变量、信号量、原子操作
**产出**：线程池库

### Week 7-8：调试工具链

**学什么**：gdb、valgrind、perf、objdump、readelf
**产出**：内核工程师调试命令速查表
